Filtering methods

  • first() : target first element of the document or target first element of the specified element
  • last() :target last element of the document or target last element of the specified element
  • eq() : target perticular element, we can give negative sign inside the eq() method which represent last element
  • filter() : target perticular element, here we can set the id , class or tag namse of the element
  • not() : target all the remaining elements which are not set by any id or class name.
  • slice(1) : slice method slice others elements and apply perticular effects, here 1 means from starting point upto last
  • slice(1,5): here starting from 1 index upto 4th position index element. 5th element is excluded here
