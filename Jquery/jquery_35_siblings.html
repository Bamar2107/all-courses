Siblings methods

  • siblings() : target siblings elements
  • siblings('.classname or #idname') : target perticular sibling
  • next() :target next sibling
  • prev() : target previous sibling
  • nextAll() : target all siblings afterwards
  • nextUntil() : target next siblings which lies between source sibling & targated sibling
  • prevUntil() : target previous siblings which lies between source sibling & targated sibling
