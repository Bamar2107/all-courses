Button class in Bootstrap

button classes

  • btn-primary = blue
  • btn-secondary = gray
  • btn-success = green
  • btn-danger= red
  • btn-warning = yellow
  • btn-info = teal
  • btn-light = light
  • btn-dark = black
  • btn-link= white background

There is one more class btn-outline class which create a outline of a button. Above all classes are place with btn-outline class like btn-outline-warning, btn-outline-success etc

for large button size btn-lg and for small btn-sm class is there. btn-block class will cover whole width of the div or container

active & disabled are the other two classes of button class
