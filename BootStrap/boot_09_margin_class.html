there are different kinds of margin avliable in bootstrap

margin(m), margin-left(ml), margin-right(mr), margin-top(mt), margin-bottom(mb)

m-auto means margin from both left right & top and bottom side.

margin ranges from 0 to 5

  • 0 means 0rem
  • 1 means 0.25rem
  • 2 means 0.5 rem
  • 3 means 1rem
  • 4 means 1.5rem
  • 5 means 3rem

m-2 means 0.5rem margin from everyside

we can set negative margin for the element, syntax is mt-md-n3 it means from top position there will be negative margin of 1rem. we can set negative margin for any breakpoints i.e, for any screen size & for any direction also.
