thead-dark, thead-light for make heading dark or light

table-hover is the class to apply some dark color when user hover on perticular row

table-borderless class to remove border on table and table-bordered used to apply border on table

table-sm to make table little bit small.

we can set different colors on table which will apply on tr tag. colors like table-primary, table-info, table-success etc.

table-responsive is the class to make table responsive for different size screen
