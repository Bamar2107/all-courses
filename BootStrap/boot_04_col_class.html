w-100 is the predefined class which breaks the columns line into the new line or you can say into next line

row-cols-2 is the predifined class which makes the columns separate according to the number of columns given. we can give any number of columns to seprate.

difference between w-100 & row-cols is, w-100 is manually break the columns & in row-cols we can set the number of columns to be separate

There are different media screen were written for different screen size in the bootstrap. The screen sizes are

extra large (col-xl), large(col-mg), medium(col-md), small(col-sm) & extra small

in the above paragraph if perticular col group is sprcified, then width of the smaller screen size col group will became max-width = 100%

Otherwise width is width=100% for same & above screen size col group.
