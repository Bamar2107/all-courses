For creating clickable button with dropdown list we need three files which we have to link to our project i.e, jquery file, proper file (3rd party file) & bootstrap js file

after linking all these above files we need to create a button with class name dropdown-toggle & data-toggle=dropdown.

After that we need to create a class called dropdown-menu

Inside that class we can create any numbers of anchor tag or other tag as per our recruitment & pass class name as dropdown-item

And if we want to create split button then before createing a button we need to create another normal button, and inside dropdown-toggle class we need to pass another classname i.e, dropdown-toggle-split
