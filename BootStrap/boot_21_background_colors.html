There are some classes for background color in bootstrap

  • bg-primary = blue
  • bg-secondary = gray
  • bg-success = green
  • bg-danger= red
  • bg-warning = yellow
  • bg-info = teal
  • bg-light = light gray
  • bg-dark = black
  • bg-white= white
  • bg-transparent = no colors

same for text colors also like text-success, text-danger etc

also we have some other classes for text like :

text-light, text-dark, text-white, text-body, text-muted, text-black-50, text-white-50

Apart from this we can set bootstrap others colors using variable
