order class is the predefined class in bootstrap to rearrange the element according to their position. order-first & order-last is the only two position class of order-class.

if we specify any screen size in order class then it will work until that class only. Once it reached smaller screen elements will go back to their original position.

we can set random position for the elements apart from just first & last position. For that we have different class in bootstrap i.e, order-0 to order-12 class which place the specific element to the thier given order number.
